Style Guide

This is the live component and visual-system reference for Ally Cui’s portfolio. It documents the current site styles, components, and accessibility patterns as they are used in the migrated HTML pages.

✦ Portfolio Ethos

Designer of clear, playful systems.

The portfolio uses clear hierarchy, editorial typography, structured grids, warm contrast, and restrained interaction to present UI/UX, mobile product, game UX, and creative technology work without turning the site into a generic template.

✦ Foundations

Layout and Spacing

The live site uses `.container` with `--container-max` and `--container-pad`, twelve-column CSS Grid for hero and case-study structures, Flexbox for navigation, buttons, metadata rows, and footer link stacks, and real media queries at 900px, 768px, and 480px to collapse grids and expose the mobile navigation toggle.

✦ Notes

Responsive and Accessibility Notes

01

CSS Grid is used for twelve-column hero, project-card, case-study, footer, and media layouts; Flexbox is used for navigation, buttons, metadata rows, contact links, and footer link groups.

02

Media queries at 900px, 768px, and 480px collapse grids, convert the primary navigation into a keyboard-accessible toggle, and make buttons full-width on small screens.

03

Every migrated page uses a skip link, visible `:focus-visible` styles, meaningful image alt text, and reduced-motion support for decorative marquee and floating animation.

04

YouTube media uses preview cards plus same-page transcript/details components when captions cannot be confirmed.
