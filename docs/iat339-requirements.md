## HTML and Accessibility

* [ ] Every page uses `<html lang="en">`
* [ ] Every page has one meaningful `<h1>`
* [ ] Heading hierarchy is logical
* [ ] Semantic HTML5 structure is used appropriately
* [ ] ARIA landmarks or labels are used where needed
* [ ] Meaningful images have appropriate `alt` text
* [ ] Decorative images use `alt=""`
* [ ] Links are recognizable and keyboard focusable
* [ ] All interactive elements show visible focus states
* [ ] All form labels are associated with their inputs
* [ ] A skip-to-content link is present on every page
* [ ] Mobile navigation works using keyboard controls
* [ ] Mobile navigation supports Escape to close
* [ ] Meaningful video or audio includes captions or a transcript
* [ ] External media links have descriptive labels

## Responsive Design

* [ ] CSS Grid is used where appropriate
* [ ] Flexbox is used where appropriate
* [ ] Real media queries are included
* [ ] Desktop layout adapts to tablet and mobile
* [ ] Navigation changes appropriately on small screens
* [ ] Grid layouts collapse without horizontal overflow
* [ ] Images resize fluidly
* [ ] Responsive images include width and height attributes
* [ ] Below-the-fold images use `loading="lazy"`

## Atomic Design and Style Guide

* [ ] `variables.css` contains design tokens
* [ ] `base.css` contains global and atom styles
* [ ] `components.css` contains reusable components
* [ ] `layout.css` contains reusable layout systems
* [ ] `templates.css` contains page-specific structures
* [ ] `responsive.css` contains media-query overrides
* [ ] Reusable component styles are not duplicated across pages
* [ ] Style Guide shows real tokens and real components from the live portfolio
* [ ] Style Guide includes HTML and CSS code samples
* [ ] Style Guide explains Atomic Design mapping

## Performance and Final QA

* [ ] No inline `style=""` attributes remain
* [ ] No `style-hover` attributes remain
* [ ] No placeholder `href="#"` links remain
* [ ] No fake contact information remains
* [ ] Images and media are optimized where practical
* [ ] Non-critical JavaScript uses `defer`
* [ ] Reduced-motion support is included
* [ ] Footer links to Style Guide, Citations, and GitHub repository work
* [ ] Site is checked around 1440px, 768px, and 390px
* [ ] Site is tested in Chrome and at least two additional browser or device contexts
* [ ] Final zip contains the completed working website
